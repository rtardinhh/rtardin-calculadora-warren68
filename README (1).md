# Calculadora de Puente Armadura Tipo Warren

Aplicación web interactiva para el análisis estructural de armaduras Warren, desarrollada con Streamlit y Python.

## Descripción

Esta herramienta permite calcular fuerzas internas, evaluar la seguridad de los materiales y visualizar el comportamiento de puentes con armadura tipo Warren de forma instantánea.

## Características

- 🏗️ Cálculo automático de reacciones y fuerzas internas
- ⚠️ Diagnóstico de seguridad con semáforo de estado
- 💡 Sugerencias inteligentes de diseño
- 📊 Tabla de miembros con esfuerzos y ratios
- 📈 Diagrama visual de la armadura
- 📜 Historial de cálculos

## Tecnologías

- Python 3.13
- Streamlit
- Pillow (para logos)
- Pandas
- Matplotlib

## Instalación

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Uso

1. Abre la aplicación en tu navegador
2. Presiona "INGRESAR A LA CALCULADORA"
3. Ajusta los parámetros en el panel lateral
4. Presiona "CALCULAR" y visualiza los resultados
